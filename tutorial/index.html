Tutorial

This webpage provides a series of tutorials that should help you to get into Rethomics. The package has a large range of applications, so you may want to read the introductory chapter and then jump to the sections that are relevant to you.

  • The ethoscope python package is installed in the ethoscope devices (raspberry pis).
  • The ethoscope_node python package will be documented soon.
  • The rethomics R package has documentation and tutorials on its own website
