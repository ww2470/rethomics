The Rethomics Package

WEBSITE UNDER CONSTRUCTION, be patient ;)

Rethomics is an R package designed to analyse high throughput behavioural data. At the moment, it focuses on analysing data from the Ethoscope platform as well as TriKinetics' Drosophila Activity Monitor.

It provides:

  • Consistent data import -- the data from different acquisition system will have the same internal structure.
  • Publication quality graph -- Integrated with ggplot.
  • A modern web-interface to control multiple devices effectively.
  • High-throughput detailed post-hock analysis -- thanks to the R package rethomics.
  • A modular design -- it is straightforward to modify both devices and software and create new experimental paradigms.
  • A high scalability -- multiple, inexpensive, ethoscopes can run in parallel within the same platform

The aim of this website is to explain how and why using rethomics to analyse large ethological data sets as well as develop a community of users.
