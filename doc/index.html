Documentation

If you want to take part in the development of Ethoscope, you will probably want to start by reading the software documentation. Different parts of the projects have different documentations:

  • The ethoscope python package is installed in the ethoscope devices (raspberry pis).
  • The ethoscope_node python package will be documented soon.
  • The rethomics R package has documentation and tutorials on its own website
