Data

As part of this project, we provide data of three types:

  • Video samples. These are references videos that can be used to test and validate tracking algorithms
  • SD cart images. These are snapshots of SD cards that can be used to replicate Ethoscope devices (see documentation).
  • .db files. These are pre-analysed results that can be used as example results of video tracking.

All these data is publicly available on our file repository.
